ortage
 
 

The REST API of the fastest gentoo package search engine in the web

1. 개요

  (2012년 6월, 적어도 제가 알고 있는 범위 내에서) 전세계 웹 기반 젠투 패키지 검색 시스템 중에서 젠투 패키지를 가장 빨리 찾을 수 있는 검색엔진입니다. PHP로 만들었기 때문에 PHP + Portage 단어를 합성하여 PHPortage라는 이름을 붙였습니다. 간단한 하이퍼텍스트 기반 검색엔진 그리고 xml과 json기반의 REST API를 통해 패키지 검색 결과를 얻어낼 수 있습니다. 세부적인 조건 설정을 통해 젠투시스템에서 사용할 수 있는 패키지가 어떤 것이 있는지 원하는 방식대로 검색 결과를 가져올 수 있습니다. 물론 검색결과에 대해 여러분이 원하시는 대로 프로그램에 넣어서 스타일링을 할 수 있습니다.

2. 지원 인터페이스

json과 xml을 지원합니다. 두가지 파일 형식에 대한 정보는 위키 페이지를 참조하시기 바랍니다.

[사용방법] : http://darkcircle.myhome.tv/phportage/phportage.(json|xml)?k=(검색단어)[&(인자 및 값들...)]*

4. 결과물의 구조

본 API에서 출력하는 결과물은 result라는 이름을 가진 노드를 루트로 두고 가장 처음에 code와 message를 출력합니다. 특별한 오류가 발생하지 않는 이상 여기서 나타나는 code는 0을 출력합니다. packages 노드가 나타나기 전의 요소들 중 실제 검색 결과 수를 나타내는 actualnumofres 를 제외한 나머지 모든 노드들은 사용자 여러분이 어떤 값을 설정했는지 다시금 확인시켜주기 위한 노드들입니다. 검색한 패키지 노드들은 packages 아래에 있으며, 여기 나타난 pkg노드는 패키지 하나에 대한 정보의 집합입니다.

이러한 결과 구조는 xml, json 둘 다 동일합니다.
